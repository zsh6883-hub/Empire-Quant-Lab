## 🛠️ 核心战术防御体系

系统的整体架构采用**双重并行清洗漏斗**，确保每一笔进入交割的资金都拥有极高的容错空间：

1. **多因子立体清洗漏斗 (STAGE 1)**
   * **流动性阻击**：筛选全市场成交量领先、具备超强承载能力的防御性资产（如京东方A、永泰能源、大唐发电等）。
   * **杠杆与估值双控**：严格限制资产负债率（$\le 65\%$），将市盈率（PE TTM）卡死在安全边界内，一票否决高风险高溢价标的。

2. **双均线动能反演雷达 (STAGE 2)**
   * 通过多周期 K 线动能追踪，当且仅当短期动态均线（MA5）强势上穿趋势基线（MA20）形成**多头黄金交叉（Golden Cross）**时，方可触发准建仓指令。

3. **Gemini 1.5 大脑 · 真AI反欺诈审计 (STAGE 3 — 核心防线)**
   * 彻底告别冰冷的关键词匹配，直接拦截市场一线的深层舆情文本。
   * 赋予 AI **“做空大师”** 极端多疑的审计逻辑，深度穿透类似“五粮液式”的渠道压货、合同负债调节利润或关联交易输送。
   * **极端容错率**：只要舆情中暗藏 1% 的暴雷或财务造假蛛丝马迹，直接实施**一票否决**，强行终止交易。

4. **安全分仓垫机制 (Portfolio Management)**
   * 杜绝全仓单打独斗，资金严密平铺至 **5 个独立持仓舱（各 20% 限额）**，单资产暴雷绝不波及系统根基。
